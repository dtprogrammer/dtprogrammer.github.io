160. Intersection of Two Linked Lists

by

Duct Tape Programmer

  • Sunday 24 Sep 2017
  • Comments
  • leetcode

Question

Write a program to find the node at which the intersection of two singly linked lists begins.

For example, the following two linked lists:

A:          a1 → a2
                   ↘
                     c1 → c2 → c3
                   ↗            
B:     b1 → b2 → b3

begin to intersect at node c1.

Notes:

If the two linked lists have no intersection at all, return null.
The linked lists must retain their original structure after the function returns.
You may assume there are no cycles anywhere in the entire linked structure.
Your code should preferably run in O(n) time and use only O(1) memory.

Solution

  • This problem can be solved by making by using two pointers.
  • pointerA traverses from headA and pointerB traverses from headB.
  • When either of them reached thee end we set them to another list.
  • Overall they will traverse the same distance and reach and the intersection point if one exists or become null at the same time in second iteration.

Time complexity

  • O (n) where n is the size of larger list

Space complexity

Constant
