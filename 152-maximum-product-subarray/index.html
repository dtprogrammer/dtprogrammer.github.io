152. Maximum Product Subarray

by

Duct Tape Programmer

  • Saturday 23 Sep 2017
  • Comments
  • leetcode

Question

Find the contiguous subarray within an array (containing at least one number) which has the largest product.

For example, given the array [2,3,-2,4],
the contiguous subarray [2,3] has the largest product = 6.

Solution

  • This problem is similar to 53. Maximum Subarray with the only difference that we need to calculate the max product.
  • In this case we need to handle the negative numbers. So we also track the minimum seen so far (the largest negative value) and when we see a negative value we switch the maEndingHere and minEndingHere since by multiplying with the current number they will switch signs and become the other way.

Time complexity

O (n)

Space complexity

Constant

Notes

  • Negative number handling is the special case here
