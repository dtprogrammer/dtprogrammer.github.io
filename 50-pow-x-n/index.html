50. Pow(x, n)

Question

Implement pow(x, n).

Solution

The problem is pretty simple although the two edge case which we need to handle are:

  1. x == 0: Leading to 1/x become infinite
  2. n == Integer.MIN_VALUE: In this case -n will overflow and come back to Integer.MIN_VALUE as its basically a Integer.MAX_VALUE + 1

Time complexity

O (log n) since we reduce n by 2 in every step

Space complexity

Constant

Notes

  1. -Integer.MIN_VALUE == Integer.MIN_VALUE
  2. Double.isInfinite() can returns true if the given number is infinitely large in magnitude
