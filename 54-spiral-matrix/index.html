54. Spiral Matrix

by

Duct Tape Programmer

  • Sunday 17 Sep 2017
  • Comments
  • leetcode

Question

Given a matrix of m x n elements (m rows, n columns), return all elements of the matrix in spiral order.

For example,
Given the following matrix:

[
 [ 1, 2, 3 ],
 [ 4, 5, 6 ],
 [ 7, 8, 9 ]
]

You should return [1,2,3,6,9,8,7,4,5].

Solution

  • The problem is straight forward. We just need to keep track of the processed rows and columns so that we avoid them in spiral traversal.

Time complexity

O (m * n)

Space complexity

Constant

Notes

  • We only go right to left and/or down to up only if the there is remaing rows and remaining columns respectively.
